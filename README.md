# Marketplace PWA

Um aplicativo de marketplace para venda de produtos novos e usados, totalmente funcional como **Progressive Web App (PWA)**. Permite aos usuários navegar, adicionar produtos, conversar via chat e destacar anúncios.

## Funcionalidades

- **Feed de produtos**: Mostra todos os produtos cadastrados, com destaque para anúncios pagos.  
- **Adicionar produto**: Cadastro rápido de produtos com imagem, descrição, preço e localização.  
- **Detalhes do produto**: Página individual para cada produto com informações completas.  
- **Perfil do usuário**: Visualize seus produtos cadastrados e informações do perfil.  
- **Chat**: Converse com outros usuários diretamente pelo app.  
- **Destaque de anúncios**: Sistema de planos pagos para aumentar o alcance do produto.  
- **PWA**: Instalável no celular, funciona offline com Service Worker.  
- **Firebase**: Banco de dados para produtos e mensagens do chat.
